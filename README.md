# EduCareer — сайт для практики

Учебный веб-сайт без CMS для публикации и поиска вакансий в образовательной среде.

## Реализовано по заданию

- Python + Flask, без CMS.
- Более 5 основных страниц: главная, вакансии, работодатели, новости, о системе, контакты, вход, регистрация, кабинет.
- Отдельная карта сайта `/sitemap`.
- Страница ошибки 404.
- Два стиля: стандартный и версия для слабовидящих.
- Роли пользователей: администратор, работодатель, студент-соискатель.
- Регистрация пользователя через форму.
- Добавление вакансии из кабинета работодателя/администратора.
- Каталог вакансий с поиском, фильтрацией по категории и городу.
- Отклики на вакансии.
- Новости/статьи и карточки работодателей.
- Форма обратной связи.
- Подготовка к размещению на бесплатном хостинге Render.

## Тестовые учетные записи

- Администратор: `admin` / `admin123`
- Работодатель: `employer` / `emp123`
- Соискатель: `student` / `stud123`

## Запуск локально

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

После запуска сайт доступен по адресу: http://127.0.0.1:5000

## Размещение на Render

1. Загрузить проект в публичный репозиторий GitHub или GitFlic.
2. На Render создать Web Service.
3. Указать команду запуска: `gunicorn app:app`.
4. После публикации вставить ссылку на сайт в отчет.
